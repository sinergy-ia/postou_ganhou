# Postou, Ganhou

Aplicacao web em Next.js para a plataforma `postou_ganhou`.

## Repositorio

- GitHub: https://github.com/sinergy-ia/postou_ganhou

## Desenvolvimento

Execute o servidor local:

```bash
npm run dev
```

Depois, abra [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
